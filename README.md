# Tapir Group — Nuxt 4 Test Task

## 1. Требования к окружению

- `Node.js` 20+ (рекомендуется LTS)
- `npm` 10+

## 2. Установка

```bash
npm i
```

## 3. Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:3000`

## 4. Production-сборка

```bash
npm run build
npm run preview
```
